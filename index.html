Rapport du TER sur la Détection de fraude bancaire

Travail Encadré de Recherche – Licence 3 MIASHS

Auteurs : Ly Amadou, Baba Compaoré, Nguyen Huy Hoang, Bryan Foulla

Résumé

Ce rapport présente une analyse de détection de fraudes bancaires via des méthodes de machine learning.

Prétraitement des données

Les données sont fortement déséquilibrées. Un rééquilibrage par oversampling et undersampling a été appliqué.

Modélisation

Les modèles XGBoost, LightGBM et CatBoost ont été testés. Le modèle le plus performant selon le F1-score est LightGBM.

Évaluation

  • Précision : 0.91
  • Recall : 0.85
  • F1-score : 0.88
  • AUC-PR : 0.93
